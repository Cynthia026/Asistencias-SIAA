Sistema de Control de Asistencias Académicas SIAA

Sistema web para el registro, consulta y administración de asistencias académicas, con interfaces diferenciadas para Alumnos, Docentes y Administradores.
El sistema permite la gestión integral de usuarios, materias, grupos y registros de asistencia mediante una estructura organizada y funcional.
